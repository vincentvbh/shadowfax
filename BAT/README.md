
This folder contains the source code of the KEM BAT implementing the parameter sets `bat-128`, `bat-257`, and `bat-769`.
Most of the files in this folder are copied from [here](https://github.com/pornin/BAT).

# Compilation

```
make all
```

For testing, run
```
./test
```

# Compile for the archive

```
make libbat.a
```
